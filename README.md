# northpass-play

  **Professional VFX media review player for iPad.**

  NorthPass Play is built by a VFX supervisor for VFX supervisors by and production teams who need
  frame-accurate playback, annotation, and colour grading during dailies and
  client review sessions.

  ## Features

  - Frame-accurate video playback with SMPTE timecode (drop-frame supported)
  - Apple Pencil annotation stored per-frame
  - Real-time LUT colour grading via Metal GPU shaders
  - Import from iCloud, Files app, or Photos
  - Client-ready PDF report export
  - Playlist and folder organisation
  - VLC-style keyboard shortcuts (J/K/L, arrow keys)

  ## Platform

  - iPad only
  - iPadOS 26+
  - Apple Pencil for annotation features or use your fingers !
